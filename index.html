Tratamiento de datos personales

UniTutor recolecta datos de identificación académica, correo institucional, rol, carrera, semestre, solicitudes de tutoría, disponibilidad, calificaciones y notificaciones para gestionar el acompañamiento académico.

La información se almacena en Firebase Authentication y Cloud Firestore, con acceso limitado a usuarios autenticados y participantes de cada tutoría. No se vende ni se comparte para fines comerciales.

Como titular puedes solicitar consulta, actualización, corrección o eliminación de tus datos mediante los canales institucionales definidos por la Universidad de Sabaneta.
